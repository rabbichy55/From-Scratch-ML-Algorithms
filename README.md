# ML Algorithms From Scratch

This repository contains my implementations of fundamental Machine Learning algorithms from scratch using Python and NumPy.

The main goal of this repo is to build a deep intuitive understanding of how these algorithms work internally, without relying on high-level ML libraries like scikit-learn.

---

## 📌 Current Contents

### Gradient Descent Variants
- Batch Gradient Descent
- Stochastic Gradient Descent (SGD)
- Mini-Batch Gradient Descent

### Linear Regression
- Simple Linear Regression
- Multiple Linear Regression
- Polynomial Linear Regression
- Accuracy / Evaluation Metrics


Each implementation is written step-by-step in Jupyter Notebooks for learning purposes.

---

## 🎯 Goals of this Repository

- Understand mathematical intuition behind ML algorithms
- Implement algorithms from scratch (NumPy only)
- Strengthen fundamentals of optimization and learning algorithms
- Build strong foundation for Deep Learning
